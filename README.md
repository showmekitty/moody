# 🎭 Moody — Дневник настроения

Приложение для отслеживания настроения, созданное на **Flutter**.

## ✨ Возможности

- 📝 Запись настроения с эмодзи (5 уровней: 😭😟😐😊😍)
- 📅 Выбор даты и времени для каждой записи
- 🏷️ Теги для категоризации (работа, спорт, друзья...)
- 📝 Текстовые заметки к каждой записи
- 🔍 Поиск по тегам
- 📊 Статистика (общая, за неделю, распределение)
- 🏆 Топ используемых тегов
- 🌙 Тёмная тема
- 📤 Экспорт всех записей в TXT файл

## 🛠️ Технологии

- **Flutter** + **Dart**
- **SQLite** (sqflite) — локальная база данных
- **Provider** — управление состоянием
- **GoRouter** — навигация
- **fl_chart** — графики
- **Material Design 3** — дизайн
- **Google Fonts** — шрифт Nunito
